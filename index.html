Model Formulations & Mathematical Insights

Ordinary Least Squares (OLS)

Linear models are solved using classical regression matrix techniques:

β = (XT X)-1 XT y

This computes the slope and intercept parameters directly without numerical gradient steps, guaranteeing global minimum loss.

Polynomial Curve Fitting

For degree d, we map feature t to dynamic Vandermonde vectors:

y ≈ β0 + β1t + β2t2 + ... + βdtd

This allows the model to capture non-linear bends, inflection points, and market momentum changes.
